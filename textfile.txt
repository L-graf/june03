Порядок ежедневной работы
1. проверка обновлений git pull (на каждую ветку)
2. проверка состояния git status
3. создание новой ветки (находимся в мастер) 
	git checkout master
	git checkout -b <новая ветка>
	если нужная ветка уже есть, команда git checkout <ИМЯ ВЕТКИ>
4. находимся в новой ветке
5. Сохранение работы git add .
	git commit -a -m "КОММЕНТАРИЙ"
6. отпривка изменнений на гитхаб
git push --set-upstream origin <ИМЯ ВЕТКИ>
если в эту ветку уже пушили, команда
git push
